Standard Library

Quill comes with a rich set of built-in functions. No imports required -- they are available everywhere.

Collections

length(x)

Returns the length of a string or list.

say length("hello")       -- 5
say length([1, 2, 3])    -- 3

join(list, separator)

Joins a list into a string with the given separator. Defaults to ", ".

colors are ["red", "green", "blue"]
say join(colors, " and ")  -- red and green and blue
say join(colors)            -- red, green, blue

split(text, separator)

Splits a string into a list by the separator.

words is split("hello world foo", " ")
say words   -- ["hello", "world", "foo"]

range(start, end)

Returns a list of numbers from start (inclusive) to end (exclusive).

say range(1, 5)   -- [1, 2, 3, 4]

sort(list)

Returns a new sorted list (ascending).

say sort([3, 1, 4, 1, 5])   -- [1, 1, 3, 4, 5]

reverse(list)

Returns a new list with elements in reverse order.

say reverse([1, 2, 3])   -- [3, 2, 1]

unique(list)

Returns a new list with duplicate elements removed.

say unique([1, 2, 2, 3, 3])   -- [1, 2, 3]

sum(list)

Returns the sum of all numbers in a list.

say sum([10, 20, 30])   -- 60

smallest(list)

Returns the smallest value.

say smallest([5, 2, 8])   -- 2

largest(list)

Returns the largest value.

say largest([5, 2, 8])   -- 8

Strings

trim(text)

Removes whitespace from both ends of a string.

say trim("  hello  ")   -- "hello"

upper(text)

Converts a string to uppercase.

say upper("hello")   -- "HELLO"

lower(text)

Converts a string to lowercase.

say lower("HELLO")   -- "hello"

replace_text(text, old, new)

Replaces all occurrences of old with new.

say replace_text("hello world", "world", "Quill")
-- "hello Quill"

startsWith(text, prefix)

Returns yes if the string starts with the given prefix.

say startsWith("hello", "he")   -- true

endsWith(text, suffix)

Returns yes if the string ends with the given suffix.

say endsWith("hello", "lo")   -- true

Math

round(n)

Rounds a number to the nearest integer.

say round(3.7)   -- 4
say round(3.2)   -- 3

floor(n)

Rounds down to the nearest integer.

say floor(3.9)   -- 3

ceil(n)

Rounds up to the nearest integer.

say ceil(3.1)   -- 4

abs(n)

Returns the absolute value.

say abs(-5)   -- 5

random()

Returns a random decimal between 0 (inclusive) and 1 (exclusive).

say random()   -- 0.7291...

randomInt(min, max)

Returns a random integer between min and max (inclusive).

say randomInt(1, 10)   -- e.g., 7

Conversion

toNumber(x)

Converts a value to a number.

say toNumber("42")   -- 42

toText(x)

Converts a value to text.

say toText(42)   -- "42"

typeOf(x)

Returns the type of a value as text: "number", "string", "boolean", "list", or "nothing".

say typeOf(42)         -- "number"
say typeOf("hello")    -- "string"
say typeOf([1, 2])    -- "list"

File I/O

File I/O functions are only available when running Quill with the CLI (quill run). They are not available in the browser playground.

read(path)

Reads the contents of a file as text.

content is read("data.txt")
say content

write(path, content)

Writes text to a file, creating or overwriting it.

write("output.txt", "Hello, file!")

append_file(path, content)

Appends text to the end of a file.

append_file("log.txt", "New entry\n")

exists(path)

Returns yes if a file exists at the given path.

if exists("config.txt"):
  say "Config found"

Date & time

now()

Returns the current date and time as an ISO 8601 string.

say now()   -- "2026-04-04T12:34:56.789Z"

today()

Returns today's date as a string (YYYY-MM-DD).

say today()   -- "2026-04-04"

Serialization

parseJSON(text)

Parses a JSON string into a Quill value.

data is parseJSON('{"name": "Alice"}')
say data.name   -- "Alice"

toJSON(value)

Converts a value to a formatted JSON string.

list are [1, 2, 3]
say toJSON(list)

System

wait(ms)

Pauses execution for the given number of milliseconds. CLI only.

wait(1000)   -- wait 1 second

exit(code)

Exits the program with the given exit code. CLI only.

exit(0)   -- exit successfully
